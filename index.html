Vue CLI 3

  1. @vue/cli — это npm-пакет, предоставляющий команду vue в терминале
  2. @vue/cli-service — это зависимость для разработки построенная на основе webpack
  3. Плагины для CLI:
    • транспиляторы, например Babel / TypeScript
    • линтеры ESLint
    • модули для тестирования
