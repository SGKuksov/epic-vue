Поддержка браузерами IE9+

Установим Vue CLI 3

npm install -g @vue/cli

Команда, чтобы проверить версию

vue --version

Заодно клонируем и установим тестовый проект

https://github.com/SGKuksov/bikes

Vue CLI 3

  • @vue/cli — это npm-пакет, предоставляющий команду vue в терминале
  • @vue/cli-service — это зависимость для разработки построенная на основе webpack
  • Плагины для CLI:
    • транспиляторы, например Babel / TypeScript
    • линтеры ESLint
    • модули для тестирования

Создадим проект

Перейдем в папку с проектами и в терминале наберем для создания нового проекта

vue create name-project

в нашем случае

vue create bikes

Выберем опции:

  1. Manually select features
  2. Babel, Router, Vuex, CSS Pre-processors
  3. Use history mode for router? Y
  4. Sass/SCSS
  5. In package.json
  6. Save this as a preset for future projects? N
  7. Use NPM
